# ARCHON BRAND GUIDE

## Brand Positioning

Archon is a technology ecosystem built around outcome-driven growth.

We help people and organizations dream without limits, learn by building, and create meaningful technology outcomes.

---

## Brand Personality

Premium  
Futuristic  
Trustworthy  
Human  
Global  
Mentor-led  
Innovation-driven  

---

## Core Message

Dream without limits.  
Build with purpose.

---

## Content Rules

Never use the term "SAP training."

Use:
- SAP mentorship
- SAP internships
- SAP career programs
- SAP consulting
- SAP project exposure
- SAP implementation experience

For non-SAP programs, the word "training" can be used.

---

## Design Direction

Dark cinematic foundation.

Electric blue and cyan highlights.

Glassmorphism.

Subtle motion.

Premium spacing.

Large typography.

High visual storytelling.
